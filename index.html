Scoring Process

Read how to build your full stack developer skills profile dashboard

Enter your individual input - Name and Second Name to start

03. Score:

Score your projects and theory elements using 1 of 4 choices on each scorecard

04. Print:

Preview 2 pages of your individual skills profile and print screen to save it

Scoring Logic

  • It takes about 20 minutes to complete scoring

  • If you leave scoring page, you will have to start the scoring from the beginnning.

  • You need to score 4 projects and 9 theory modules

  • Each theory module consists of several study elements, 87 in total

  • Average scoring % depends on practical application of the knowledge that you acquired defined by scoring choices

  • Each scorecard has 4 scoring choices (see example below)
